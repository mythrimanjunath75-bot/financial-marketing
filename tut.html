AI-Powered Personal Finance — Affordable, Personalized, Accessible

A prototype platform that combines an NLP assistant and human advisors to help Indian users make confident financial decisions. Free consultations, portfolio tools, risk analysis and practical recommendations.

Launch App How it works
Live demo
Profile onboarding
AI chat assistant
Portfolio manager

How it works

Three simple steps to get started:

1. Create profile

Enter your name, age, monthly income (capped at ₹1,00,00,000), savings and goals. For users under 18 monthly income is hidden in the dashboard.

2. AI Assistant

Ask budgeting, savings and investment questions. The embedded NLP chatbot provides contextual replies and suggests next steps.

3. Portfolio & Advisors

Add investments, view recommended limits, and book a free human advisor consultation for deeper guidance.

Design goals

Provide affordable, personalized and accessible financial advisory using AI. Limit age-dependent disclosures, cap numeric inputs (₹1 crore), and combine automated suggestions with human support.

  • Age-based UI rules (hide income if < 18)
  • Investment checks (min ₹5,000, max 20% of savings per allocation)
  • Simple, localised presentation (Indian rupees)

Quick start

To run locally:

1. Open this folder in File Explorer.
                    2. Double-click index.html (or)
                    3. Serve with Python: python -m http.server 8000 then open http://localhost:8000/
Click "Launch App" to open the interactive demo.
Prototype • Not financial advice • Data is local to your browser • © Project
